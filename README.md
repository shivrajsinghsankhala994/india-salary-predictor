# 💰 India Salary Predictor

A Machine Learning web app that predicts salaries for Indian job market.

🔗 **Live Demo:** [Click Here](https://india-salary-predictor.onrender.com)

---

## 📊 Project Overview

This app predicts expected CTC (salary) based on:
- Years of experience
- Education level
- City / Location
- Job Role
- Industry type

---

## 🎯 Model Performance

| Model | Accuracy |
|---|---|
| Linear Regression | 72% |
| Random Forest | 84% |
| Gradient Boosting | 89% ✅ |

**Final Model:** Gradient Boosting Regressor
**Accuracy:** 89% (R2 Score)
**Average Error:** ±1.3 LPA
